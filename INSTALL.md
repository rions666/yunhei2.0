# 黑名单查询系统 - 安装文档


#### 必需组件
- **Web 服务器**: Apache 2.4+ 或 Nginx 1.10+
- **PHP 版本**: PHP 7.0 或更高版本（推荐 PHP 7.4+）
- **数据库**: MySQL 5.6+ 或 MariaDB 10.0+
- **磁盘空间**: 至少 100MB 可用空间（不含上传文件）

#### PHP 扩展要求
必须启用以下 PHP 扩展：
- `mysqli` - MySQL 数据库连接
- `gd` - 图片处理
- `json` - JSON 数据处理
- `session` - 会话管理
- `mbstring` - 多字节字符串处理

#### PHP 配置要求
```ini
upload_max_filesize = 10M      # 上传文件大小限制
post_max_size = 20M            # POST 数据大小限制
max_execution_time = 300       # 脚本执行时间
memory_limit = 128M            # 内存限制
```

### 浏览器要求
- Chrome 60+
- Firefox 60+
- Safari 12+
- Edge 79+
- 或其他支持 HTML5 的现代浏览器

## 安装步骤


#### 步骤 1: 访问安装页面
在浏览器中访问：
```
http://yourdomain.com/install.php
```

#### 步骤 2: 系统检测
安装程序会自动检测：
- ✅ PHP 版本是否符合要求
- ✅ 必需的 PHP 扩展是否已安装
- ✅ 目录权限是否正确
- ✅ 数据库连接是否可用

如果检测失败，请根据提示修复问题后刷新页面。

#### 步骤 3: 填写数据库信息
在安装表单中填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| 数据库主机 | MySQL 服务器地址 | `localhost` 或 `127.0.0.1` |
| 数据库端口 | MySQL 端口号 | `3306`（默认） |
| 数据库名称 | 之前创建的数据库名 | `blacklist` |
| 数据库用户名 | 数据库用户 | `blacklist_user` |
| 数据库密码 | 数据库密码 | `your_password` |

**注意事项**:
- 数据库主机通常填写 `localhost`
- 如果 MySQL 在远程服务器，填写远程 IP 地址
- 管理员密码建议使用字母+数字+符号组合
- 请牢记管理员账号和密码

#### 步骤 4: 开始安装
1. 点击"开始安装"按钮
2. 系统会自动执行以下操作：
   - 测试数据库连接
   - 创建数据表结构
   - 插入初始数据
   - 创建管理员账号
   - 生成配置文件
   - 创建安装锁文件

#### 步骤 5: 安装完成
安装成功后会显示：
```
✅ 安装成功！
系统已成功安装，请删除 install.php 文件以确保安全。
```

点击"进入后台"按钮，跳转到登录页面。
